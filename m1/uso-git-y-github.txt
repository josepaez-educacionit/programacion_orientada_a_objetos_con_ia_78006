## **1. Inicialización de un Repositorio Local**

### **1.1 Instalación de Git**

1. **Descargar Git:**
   - Ve al sitio oficial de Git: [https://git-scm.com/](https://git-scm.com/).
   - Descarga la versión adecuada para tu sistema operativo (Windows, macOS o Linux).

2. **Instalar en Windows:**
   - Ejecuta el instalador descargado.
   - Acepta los términos y condiciones.
   - Durante la instalación, selecciona Git Bash como terminal predeterminada si prefieres usarlo en lugar de la consola de Windows.

3. **Verificar la instalación:**
   - Abre la consola de Windows o Git Bash y ejecuta:
     ```bash
     git --version
     ```
     Resultado esperado: Algo como `git version 2.x.x`.

### **1.2 Configuración Inicial**

Antes de usar Git, configura tu nombre de usuario y correo electrónico:

1. **Configurar nombre de usuario:**
   ```bash
   git config --global user.name "Tu Nombre"
   ```

2. **Configurar correo electrónico:**
   ```bash
   git config --global user.email "tuemail@example.com"
   ```

3. **Verificar la configuración:**
   ```bash
   git config --global --list
   ```
   Verás algo como:
   ```
   user.name=Tu Nombre
   user.email=tuemail@example.com

   ### **1.3 Crear un Repositorio Local**



1. **Crea una carpeta de trabajo:**
   ```bash
   mkdir C:\EducacionIT\fintech-app
   cd C:\EducacionIT\fintech-app
   ```

2. **Inicializa el repositorio Git:**
   ```bash
   git init
   ```
   Esto crea un directorio oculto `.git` donde se almacena la información del repositorio.


   ## **2. Creación de una Cuenta en GitHub**

1. **Acceso a GitHub:**
   - Ve a [https://github.com](https://github.com).

2. **Registro:**
   - Haz clic en "Sign up".
   - Ingresa tu correo electrónico, elige un nombre de usuario y una contraseña.

3. **Configuración Inicial:**
   - Sigue las instrucciones para confirmar tu correo.
   - Completa los pasos para personalizar tu perfil.



## **3. Vinculación del Repositorio Local con un Servidor Remoto**

### **3.1 Crear un Repositorio en GitHub**

1. En GitHub, haz clic en "New Repository".
2. Ingresa el nombre: `fintech-app`.
3. Selecciona la opción "Public" y haz clic en "Create Repository".



2. **Verificar configuración remota:**
   ```bash
   git remote -v
   ```
   Verás algo como:
   ```
   origin  https://github.com/MiUsuario/fintech-app
   (fetch)
   origin  https://github.com/MiUsuario/fintech-app
   (push)
   ```

### **3.3 Realizar un Push Inicial**

1. **Preparar los archivos iniciales:**
   ```bash
   echo "# fintech app" > README.md
   git add README.md
   git commit -m "Primer commit"
   ```

2. **Enviar los cambios al repositorio remoto:**
   ```bash
   git push -u origin main
   ```

   ## **4. Flujo de Trabajo en Git**
1. **Verificar el estado:**
   ```bash
   git status
   ```
   Muestra archivos modificados, preparados y no rastreados.
